# Cartion Mockup

Luxury-inspired static mockup for the jewelry maison **Cartion**, drawing on monochrome Cartier aesthetics.

## Quick start
- Open `index.html` in your browser.
- Edit `styles.css` to adjust typography, spacing, or palette.

## Notes
- Fonts are loaded from Google Fonts: Cormorant Garamond (serif) and Manrope (sans).
- Palette: black/white with a muted gold accent (`#c6a86d`).
